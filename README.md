# 🔐 Password Generator & Strength Checker

A simple Python project that generates secure random passwords and checks their strength based on common security standards.

## 📌 Project Overview

This project allows users to:
- Generate a random password of a custom length.
- Check the strength of the generated password.
- Learn the importance of creating secure passwords.

This project was completed as **Task 2** of the **CodeVedx Python Internship**.

---

## ✨ Features

- 🔑 Generates random passwords
- 🔢 Supports custom password length
- 🔠 Includes uppercase and lowercase letters
- 🔢 Includes digits
- 🔣 Includes special characters
- 📊 Evaluates password strength
- 💻 Simple and beginner-friendly Python code

---

## 🛠️ Technologies Used

- Python 3
- random module
- string module

## 💻 Sample Output


## 📖 Password Strength Criteria

The password strength is determined based on:

- ✅ Minimum length of 8 characters
- ✅ Contains uppercase letters
- ✅ Contains lowercase letters
- ✅ Contains digits
- ✅ Contains special characters

### Strength Levels

| Score | Strength |
|--------|----------|
| 0–2 | Weak |
| 3–4 | Medium |
| 5 | Strong |
